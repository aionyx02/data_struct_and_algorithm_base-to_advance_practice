# AVL Deletion Tree

Implement an AVL tree with unique integer keys, stored heights, parent links,
insertion, and deletion rebalancing.

Input starts with `operationCount`. Operations are `insert key`, `erase key`,
`contains key`, `root`, `height key`, `balance key`, `preorder`, `inorder`,
`size`, and `clear`.

Successful insertion prints nothing. Successful deletion prints `REMOVED`;
duplicates print `DUPLICATE`; missing deletion and node queries print
`NOT_FOUND`. Empty tree views print `EMPTY`.

Do not use standard containers.
