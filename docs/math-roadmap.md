# 數論與數學學習路線

## 0. 定位與範圍

這是一份**個人數學學習路線**，是 `docs/algorithm-roadmap.md` 的數學專線，
聚焦競賽選手必備的數論、組合、代數、多項式、機率與幾何數學。

- **範圍同 ADR-0006（proposed）**：與演算法路線一樣，本線在 ADR-0006 被開發者
  標記為 `accepted` 前，**不為本 repo 新增 Judge 題**；練習在外部平台
  （Library Checker、Project Euler、Codeforces Math、AtCoder、洛谷數學區）進行。
- 每階段標 **🧪 Judge 候選**：說明落地為本地 Judge 題時的差分 oracle 策略。數學題
  多為**取模整數唯一輸出**，天然適合精確差分；少數（高精度浮點、隨機質數測試）需
  後續 ADR。
- 目標：補齊「所有職業選手都會」的數學知識體系，從取模運算到 Min_25 篩、多項式全家桶、
  鞅與停時定理，支撐 `algorithm-roadmap.md` 的 G6/G7（基礎）與 G10/G13（進階）。

## 1. 難度分級

沿用演算法路線的 L0–L6（見 `docs/algorithm-roadmap.md` §1）：L0 套模板，
L3 能證明與轉化，L5 非模板解，L6 國際賽壓軸（重型數學模板 + 縫合）。

## 3. M0 取模運算與基礎（約 25 題，L0–L2）

- 取模四則、負數取模、`(a*b)%p` 溢位防護（`__int128` / 龜速乘 / Barrett / Montgomery）。
- 快速冪、快速乘、矩陣快速冪雛形。
- 整數性質：整除、餘數、進位制轉換、二進位與位元運算。
- 取模逆元的三種來源預告（費馬、擴展歐幾里得、線性遞推，於 M1/M2 展開）。
- 🧪 Judge 候選：輸出唯一整數，直接差分；生成器 pin 住會溢位的大數前綴。

驗收：能在 `p ~ 10^18` 下安全做模乘，並用快速冪算 `a^b mod p`。

## 4. M1 數論基礎（約 40 題，L1–L3）

- GCD / LCM、歐幾里得與**擴展歐幾里得**、裴蜀定理、線性同餘方程 `ax≡b (mod m)`。
- 質數：試除、埃氏篩、**線性（歐拉）篩**；質因數分解、因數個數 / 因數和。
- 大數判質與分解：Miller–Rabin、Pollard's Rho。
- 模逆元：擴展歐幾里得求逆、`1..n` 線性求逆元、階乘逆元。
- 數論函數初步：`φ`（歐拉函數）、`d`（因數個數）、`σ`（因數和）的篩法計算。
- 🧪 Judge 候選：分解 / 篩結果唯一 → 差分；Miller–Rabin / Rho 為隨機化，oracle 用
  確定性試除在小範圍交叉驗證，大範圍只比對「是否質數 / 任一非平凡因數」。
- 銜接：擴展歐幾里得 ↔ M2 CRT；線性篩 ↔ M3 積性函數。

驗收：能線性篩出 `φ`、最小質因數，並用 Pollard's Rho 分解 `~10^18` 的數。

## 5. M2 同餘理論（約 35 題，L2–L4）

- 中國剩餘定理（CRT）與**擴展 CRT**（模不互質）。
- 費馬小定理、歐拉定理、降冪（歐拉降冪 / 擴展歐拉定理）。
- 乘法階、原根存在性與求法、指標（離散對數的代數基礎）。
- 離散對數：BSGS / 擴展 BSGS（`a^x≡b`，含 gcd(a,p)≠1）。
- 二次剩餘：勒讓德 / 雅可比符號、Tonelli–Shanks 開平方根；高次剩餘概念。
- 🧪 Judge 候選：唯一整數 / 「無解」分支 → 差分；pin 住無解與 mod 不互質的邊界前綴。
- 銜接：原根 ↔ M7 NTT 模數選擇；BSGS ↔ M11 哈希。

驗收：能用 exCRT 合併不互質同餘、用 BSGS 解離散對數、用 Tonelli–Shanks 開模意義平方根。

## 6. M3 積性函數與篩法（約 35 題，L3–L6）

- 積性 / 完全積性函數、Dirichlet 卷積、單位元與逆。
- 莫比烏斯函數 `μ`、**莫比烏斯反演**（兩種形式）、整除分塊（數論分塊）。
- 線性篩任意積性函數；前綴和的整除分塊 `O(√n)` 求和。
- 亞線性篩：**杜教篩**（`O(n^{2/3})` 求積性函數前綴和）。
- **Min_25 篩 / 洲閣篩**：`O(n^{3/4}/log n)` 求積性函數前綴和（質數處為多項式）。
- Powerful Number 篩；亞線性篩在 `φ`、`μ`、`d`、`σ` 前綴和的套用。
- 🧪 Judge 候選：前綴和唯一整數 → 強差分；用線性篩暴力前綴和作 oracle 對拍杜教 /
  Min_25 版本（小 n 全等，大 n 抽樣點驗證）。屬 L6 重型模板，適合高分值 Judge 題。
- 銜接：莫反 + 整除分塊 ↔ G10 計數 DP；杜教篩 ↔ M12 估計。

驗收：能用整除分塊 + 莫反化簡求和式，並用杜教篩 / Min_25 在亞線性時間求積性前綴和。

## 7. M4 組合數學（約 45 題，L1–L4）

- 排列組合、二項式定理與常見恆等式（吸收 / 範德蒙德 / 曲棍球棒）。
- 組合數取模：階乘逆元、**Lucas**、**擴展 Lucas**（模非質數）。
- **容斥原理**、二項式反演、子集反演、最值反演（min–max 容斥）。
- 卡特蘭數、那羅延數、超級卡特蘭 / Motzkin；格路徑計數與反射容斥。
- 第一 / 第二類斯特林數、貝爾數、整數拆分（五邊形數定理）、錯排。
- 🧪 Judge 候選：計數（取模）唯一 → 強差分；用 `O(2^n)` / `O(n^2)` 暴力枚舉作 oracle
  對拍封閉公式，pin 住會踩 Lucas / 容斥邊界的小參數。
- 銜接：容斥 ↔ G10 計數 DP；卡特蘭 ↔ 棧 / 括號（Stage F 線性結構）。

驗收：能用 Lucas / exLucas 算大組合數取模，能用容斥 / 反演把計數題化簡。

## 8. M5 生成函數（約 25 題，L3–L5）

- 普通生成函數（OGF）、指數生成函數（EGF）與其組合意義（標號 / 非標號）。
- 形式冪級數運算、封閉形式、有理生成函數與線性遞迴對應。
- 牛頓恆等式、對稱多項式；**拉格朗日反演**（求複合逆係數）。
- 生成函數解組合計數（拆分、樹計數、卡特蘭族的統一視角）。
- 🧪 Judge 候選：係數（取模）唯一 → 差分；常與 M7 多項式運算合併落地為「求第 k 項」。
- 銜接：OGF ↔ M7 多項式求逆；EGF ↔ M9 群論計數。

驗收：能把計數問題寫成生成函數並用封閉形式 / 拉格朗日反演取出係數。

## 9. M6 線性代數（約 35 題，L2–L5）

- 高斯消去：實數 / 模意義 / 異或（線性基）；解線性方程組與判秩。
- 行列式：模質數行列式、輾轉相除求模非質數行列式；矩陣求逆。
- 矩陣快速冪：線性遞迴加速；轉移矩陣建模。
- **Berlekamp–Massey**：由前若干項求最短線性遞迴；配合 Kitamasa / 多項式取模求第 n 項。
- 矩陣樹定理（生成樹計數）、BEST 定理（歐拉迴路計數）、LGV 引理（不相交路徑計數）。
- 特徵多項式、Cayley–Hamilton 應用。
- 🧪 Judge 候選：解 / 行列式 / 計數唯一（取模或有理）→ 差分；線性基 / 矩陣樹用暴力枚舉
  小圖對拍。
- 銜接：線性基 ↔ 異或問題（G10）；矩陣樹 ↔ G9 圖計數。

驗收：能用高斯消去解模意義方程、用矩陣樹定理數生成樹、用 BM + Kitamasa 求遞迴第 n 項。

## 10. M7 多項式全家桶（約 40 題，L4–L6）

- **FFT**（浮點）、**NTT**（質數模 + 原根）、**任意模數 NTT（MTT，三模 / 拆係數）**。
- 多項式：乘法、求逆、除法 / 取模、開根、`ln`、`exp`、快速冪。
- 多點求值與快速插值、分治 NTT、多項式複合與複合逆（拉格朗日反演實作）。
- 應用：卷積型計數 DP 加速、字串 / 母函數係數提取、分治建構。
- 常數最佳化：三次變兩次 FFT、預處理單位根、模數選擇。
- 🧪 Judge 候選：係數序列唯一（取模）→ 強差分；用 `O(n^2)` 樸素卷積作 oracle 對拍，
  pin 住 NTT 友善 / MTT 邊界模數。屬 L6 重型模板，適合 Library-Checker 風格 Judge 題。
- 銜接：NTT ↔ M5 生成函數、G10 卷積 DP；多點求值 ↔ M6 特徵多項式。

驗收：能用 NTT / MTT 在 `O(n log n)` 完成多項式乘法 / 求逆 / `exp`，並應用於計數。

## 11. M8 機率與期望（約 35 題，L2–L5）

- 古典 / 條件機率、全機率與貝氏；期望的**線性性**（最重要工具）。
- 期望 DP、機率 DP；狀態設計與無後效性（與 G10 互補）。
- 馬可夫鏈、吸收機率與期望步數（解線性方程，接 M6）。
- **鞅與停時定理（optional stopping）**、勢函數法求期望步數。
- min–max 容斥求期望、隨機變數的方差與獨立性。
- 🧪 Judge 候選：期望多為**有理數取模**（分數 mod p）→ 唯一可差分；用小狀態暴力期望
  （解線性系統）作 oracle 對拍。
- 銜接：吸收鏈 ↔ M6 高斯消去；min–max 容斥 ↔ M4。

驗收：能用線性性 / 鞅 / min–max 容斥避開硬解，能把期望題化為線性方程或 DP。

## 12. M9 群論計數（約 20 題，L3–L5）

- 置換群、軌道–穩定子定理、**Burnside 引理**、**Pólya 計數定理**。
- 循環指標多項式、項鍊 / 染色 / 同構計數。
- 商集與對稱性建模；與 EGF（M5）結合的標號計數。
- 🧪 Judge 候選：計數唯一（取模）→ 差分；用暴力枚舉所有等價類作 oracle（小規模）。
- 銜接：Pólya ↔ M4 組合、M5 EGF。

驗收：能用 Burnside / Pólya 處理旋轉 / 翻轉對稱下的染色計數。

## 13. M10 賽局論（約 25 題，L2–L5）

- 公平組合賽局、必勝 / 必敗態、Nim 與 **Nim 和（異或）**。
- **Sprague–Grundy 定理**、SG 函數、遊戲和（多個子遊戲 SG 異或）。
- 階梯 Nim、翻硬幣遊戲、Green Hackenbush（樹 / 一般圖）。
- 反 Nim（misère）、Multi-SG、打表找 SG 規律。
- 超現實數 / 非公平賽局概念（進階）。
- 🧪 Judge 候選：勝負 / SG 值唯一 → 差分；用記憶化搜尋暴力 SG 作 oracle 對拍規律解。
- 銜接：SG 打表 ↔ G3/G10 DP；Nim 和 ↔ 二進位。

驗收：能把組合賽局拆成子遊戲、求 SG 並用異或判勝負，能處理 misère 修正。

## 14. M11 變換與數值（約 30 題，L3–L6）

- 位元卷積：**FWT / FMT**（或 / 與 / 異或卷積）、子集和變換、子集卷積。
- 快速沃爾什 / 莫比烏斯（高維前綴和 SOS，與 G10 互通）。
- 自適應辛普森積分、數值積分與求根（牛頓法、二 / 三分）。
- 線性規劃：單純形法、對偶；網路流–LP 對偶直覺（接 G9）。
- 拉格朗日乘數 / 凸優化（Aliens / WQS，接 G10）、凸包技巧的代數視角。
- 高精度（大數）四則與 FFT 乘法、進位制與分數類。
- 🧪 Judge 候選：FWT / 高精度為整數唯一輸出 → 差分；單純形 / 數值積分為浮點 → 需後續
  ADR（容差比對），建議先落地整數類（FWT、高精度）。
- 銜接：FWT ↔ G10 子集 DP；單純形 ↔ G9 流。

驗收：能用 FWT 做三種位元卷積、用自適應辛普森算定積分、用單純形解小型 LP。

## 15. M12 解析數論與估計（約 15 題，L4–L6）

- 素數計數 `π(n)`（Meissel–Mertens / Lucy_Hedgehog / Min_25 質數部分）。
- 素數定理、調和級數 `~ln n`、因數函數均值、`φ`/`μ` 的漸近估計。
- 階乘 / 二項式的位數與質因數冪次（勒讓德公式 / Kummer 定理）。
- 連分數、Stern–Brocot 樹、Pell 方程、最佳有理逼近。
- 🧪 Judge 候選：`π(n)`、質因數冪次唯一 → 差分；小範圍用線性篩計數作 oracle。
- 銜接：`π(n)` ↔ M3 Min_25；連分數 ↔ M1 擴展歐幾里得。

驗收：能在 `n ~ 10^{11}` 求 `π(n)`，能用勒讓德 / Kummer 算階乘與組合數的質因數冪次。

## 16. 數學能力驗收

- **基礎（M0–M4，對應 G6）**：能在取模環境正確處理逆元、組合數、容斥與線性遞迴；
  標記 `Math Foundation Ready`。
- **進階（M5–M11，對應 G7/G10）**：能用 NTT/MTT、杜教/Min_25 篩、生成函數、矩陣樹、
  期望鞅、FWT 解中高難題；標記 `Math Advanced Ready`。
- **頂尖（含 M12 與重型模板縫合）**：能在 WF/IOI 壓軸把數學（多項式 / 篩 / 計數）與
  資料結構、DP 縫合並證明複雜度；標記 `Math International Ready`。
