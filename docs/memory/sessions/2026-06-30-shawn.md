## Detailed Notes

- M40 solves `(A + u v^T)x = b` modulo `1000000007` using the
  Sherman-Morrison identity. Queries in a batch are independent and reuse the
  original invertible matrix `A`.
- The stress oracle biases early generated cases toward a non-symmetric matrix
  and a denominator-sensitive identity case so the two known-wrong submissions
  fail deterministically under the standard 4-case stress CTest seed.
- M41 answers `det(A + U V^T)` through `det(A) * det(I + V^T A^-1 U)`.
  Its fixed and stress cases catch both `U^T A^-1 V` core orientation mistakes
  and implementations that multiply only the low-rank core diagonal.
- The first M41 targeted run found an incorrect expected value in
  `04-hidden-sequence.out`; the second query uses base `A = [1]`, so the answer
  is `1 + 4*6 = 25`, not `29`.
- M42 answers block determinants using
  `det([A B; C D]) = det(A) * det(D - C A^-1 B)`. Its fixtures catch both the
  sign error in the Schur complement and the transposed-product mistake
  `B^T A^-1 C^T`.
- M43 solves `(A + U V^T)x = b` with the Woodbury identity. Its fixed and stress
  cases catch transposed low-rank core orientation and diagonal-only core solves.
- The first M43 targeted run found an incorrect expected value in
  `04-hidden-sequence.out`; the second query uses base `A = [1]`, so the answer
  is `7 / 25 = 960000007`.
- M44 solves `[A B; C D][x; y] = [p; q]` with the Schur complement. Its fixed
  and stress cases catch sign errors in both the complement and reduced RHS, plus
  the transposed rectangular-product mistake.
- M45 prints the inverse of `[A B; C D]` using the same Schur complement. Its
  fixed and stress cases catch plus-sign complements and transposed rectangular
  products in the block inverse formula.
- M46 computes `rank([A B; C D]) = n + rank(D - C A^-1 B)`. Its fixed and
  stress cases include singular Schur complements, `k > n`, plus-sign traps,
  and transposed rectangular-product traps.
- The broader `TASK.CURRICULUM.020` remains in progress; no completed marker was
  added for the whole task.
