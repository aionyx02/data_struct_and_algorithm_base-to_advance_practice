## Stage A2 Segment Tree Beats Batch

- Added `A20-segment-tree-beats` (D5) as the twelfth and final Stage A2 problem:
  a Ji Driver (Segment Tree Beats) tree for range `chmin` with range `sum` and
  range `max`. Each node tracks max, strict second max, count of max, and sum;
  `chmin` skips when `x >= max`, applies in `O(1)` when `secondMax < x < max`,
  and otherwise recurses into both children.
- Created `problem.json`, `statement.md`, four fixed test pairs (`01-public`,
  `02-boundary`, `03-beats-second-max-invariant`, `04-hidden-sequence`), one
  correct fixture, and two known-wrong fixtures: `..._no_recurse.cpp` (applies
  the tag on any covered node, skipping the Beats recursion) and
  `..._full_count.cpp` (scales the sum drop by the whole node size instead of
  the max count).
- Added `generateSegmentTreeBeats` (oracle = brute array). The fixed prefix
  pins indices 0 and 1 (always a left-spine node) to `2000` over `1000`, then
  `chmin 0 1 1500` (between second max and max -> exposes the full-count bug) and
  `chmin 0 1 500` (at/below the second max -> exposes the no-recurse bug), so
  both wrong fixtures fail on the first generated case regardless of seed.
  Registered the generator and wired catalog, Judge, and stress CTest entries.
- Authoring slip fixed during verification: `02-boundary.in` initially omitted
  the initial-values line, which crashed every submission with empty output;
  added the missing `9` value line.
