## Detailed Notes

Graph exercises expose representation invariants directly through matrix rows,
ordered neighbor views, degree counters, edge counts, and reusable edge-table
slots. Shortest paths, MST, topological sorting, and other graph algorithms
remain deferred.

Hashing exercises expose exact table layouts and probe counts so that a
submission cannot pass by implementing set membership with a different hidden
representation. Tombstone state, node-pool slots, load factor, and capacity
changes remain directly observable.

Search-tree exercises expose rotations, parent links, stored heights, balance
factors, node occupancy, colors, and black height. The final Basic Concepts
problems expose exact shift and copy costs while verifying that ADT behavior
survives a representation change.
