## COMPLETED: TASK.CURRICULUM.020 - M3 first batch (M10-M13)

Added four prerequisite-ordered number-theory problems under
`problems/math/number-theory/`, each with `problem.json`, `statement.md`, four
fixed test pairs, one correct + two known-wrong submissions, a trusted
differential generator, and catalog/judge/stress CTest wiring.

- **M10-mobius-divisor-block** (D3): `mu`, `mu_prefix`, and `floor_sum` (divisor
  summatory function via 整除分塊). Bugs: dropped `+1` in the block count
  (`floor_off_by_one`); `mu` not zeroed on non-squarefree numbers
  (`not_squarefree`).
- **M11-multiplicative-sieve** (D3): one linear sieve yielding `phi`, `mu`, `d`,
  `sigma`. Bugs: divisor count doubled in the `p | i` branch
  (`divisor_pp_branch`); `phi` multiplied by `(p-1)` instead of `p` on prime
  powers (`phi_prime_power`).
- **M12-mobius-inversion-gcd** (D3, mod 1e9+7): `coprime` and `gcd_sum` double
  sums via μ / φ + 2D divisor blocking. Bugs: products overflow before the mod
  (`missing_mod`); block step ignores `m` (`single_step`).
- **M13-du-sieve** (D4): `phi_prefix` / `mu_prefix` for n up to 1e9 via the Du
  sieve (linear-sieve threshold 1.5e6 + memoised recurrence + divisor blocks).
  Bugs: wrong harmonic base `n(n-1)/2` (`phi_base`); dropped `+1` block term
  (`block_off_by_one`). Bug-trapping fixed tests use n above the threshold so
  the recurrence is exercised.
